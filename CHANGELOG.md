# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added
- Initial public release
- User authentication with ReCaptcha protection
- Postcard registration (sent/received)
- Photo uploads for postcards and stamps
- Interactive maps (Google Maps & AMap)
- Geocoding service for automatic coordinates
- Public gallery for sharing collection
- Private galleries for postcards and stamps
- Statistics dashboard
- Contact management with auto-fill
- Arrival confirmation system with email notifications
- Multi-currency support with IDR conversion
- Responsive vintage-themed design

### Security
- Route protection with authentication middleware
- Data ownership verification on all queries
- Secure file uploads using Laravel Storage facade
- Environment-based configuration
